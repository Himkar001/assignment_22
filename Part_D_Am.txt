“Explain how to perform data analysis using Pandas and visualization
using Matplotlib with examples.”

------------------------------------------------------------------------

2.  AI Output:

Data analysis using Pandas involves loading data, cleaning it, selecting
relevant columns, filtering rows, and computing statistics.

Example:

import pandas as pd import matplotlib.pyplot as plt

df = pd.read_csv(“data.csv”)

print(df.head())

print(df.describe())

filtered = df[df[‘price’] > 5000]

Visualization using Matplotlib helps in understanding patterns and
distributions.

Example:

plt.hist(df[‘price’]) plt.title(“Price Distribution”) plt.show()

df[‘price’].plot() plt.title(“Price Trend”) plt.show()
